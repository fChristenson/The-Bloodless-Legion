# Hildo's

## Description	
The shop is a single storey stone-walled building, with a gray tile roof and a smooth stone floor. A collection of sundered shields hangs from the walls.

## Shopkeeper	
The shopkeeper is a stout male halfling named Hildo Grubburr. He is a poor haggler.
